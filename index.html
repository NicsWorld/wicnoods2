Dinner.dev

Dinner.dev is a project that was born from my wife and I asking each other, "What's for dinner?" And never being able to come to a conclusion. This app uses your location to randomly select a restaraunt near you.

I'm still looking forward to adding more features and improving this project in the future, such as a voting systems, and a different variety of filters.

Check it out here! https://dinner.dev

Urls.day

urls.day is a project that I launched strictly to learn. I wanted to learn more about the Golang programming language, which I wrote the backend in, and I wanted to learn more about Google Cloud, which I deployed this application to.

This application is essentially a url shortener. You can shorten, lengthen (sp?), and custom endings to a url.

It was a lot of fun to build, and I learned a lot through this project: https://urls.day

Diddle chrome extension

Diddle was a fun little experiment of mine, that was inspired by Slack. When you are in a Slack huddle, when someone is sharing their screen, you can essentially doodle on their screen and it will slowly fade away.

I was inspired enough to see if I could find a way to accomplish this on the web. It was a lot of fun, and I am proud I was able to figure out *something* that works.

It's not currently in the Chrome Extension store, but you can install it locally and play with it if you wish: https://github.com/NicsWorld/diddle
